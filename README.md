Simple Guide to accessing Donlisa Recharge and bills payment Apis

// register new user 
// using post request

``` bash
donlisa.com/api/register

-name
-email
-phone
-password
```

// login a new user

``` bash
donlisa.com/api/register

-email(this field can be email or phone but end the value with the {email} property)
-password
```
